코로나 창궐 후 시작한 첫 학부 프로젝트

여러 백신및 확진자 정보를 보여주는 어플리케이션을 자바 스윙으로 구현

🗓️ **작업기간** : 2021/10.6 → 2022/10.23

👨‍💻 **투입인원** : 3명

📒 **주요업무** 

- 기능구현
- 크롤링으로 지역별 확진자정보를 가져옴 및 ui에서 조회와 관리자 로그인 부분을 담당
- 버그 수정 및 깃관리

🌱 **스킬 및 사용툴**
